1- Syntax Difference

Normal Function Syntax

function function-name(parameters){
//code
}

Arrow Function Syntax

const function-name= (parameters)=>{
//code
}

2- Argument Binding

Normal Function

In regular function, Arguments keywords can be used to access the arguments of which passed to function.

Arrow Function

Arrow functions do not have an arguments binding. if you want to access arguments in an arrow function, you can use the rest operator

3- This Operator

Normal Function

In Normal Function, We can use "this" operator

Arrow Function

In Arrow function, "this" operator can't be used.

4- new keyword

Normal Function

Regular functions are constructible, they can be called using the new keyword.

Arrow Function

Arrow functions can never be used as constructor functions. Hence, they can never be invoked with the new keyword

4- code understanding

Normal Function

Regular functions are less efficient as compared to the arrow function

Arrow Function

Arrow functions increase code readability and makes the code clean as comapred to normal function
